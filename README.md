# nya_engine

NyaEngine : 3D Game Engine written in [crystal](https://crystal-lang.org/) \[**Work in progress**\]

Inspired by Unity3D and X-Ray Engine

# How to test

Install dependences :
* `crystal`
* `libLibSDL2-dev`
* `libpangocairo-dev`
* `libode-dev` - for ODE bindings

```sh
  git clone https://github.com/unn4m3d/crystal_sdl_test/
  cd crystal_sdl_test
  crystal deps
  crystal ./src/main.cr
```
