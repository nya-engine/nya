# nya_engine

<s>NyaEngine : 3D Game Engine written in crystal</s> just a SDL bindings test

# How to test

Install dependences :
* `crystal`
* `libsdl2-dev`
* `libpangocairo-dev`
* `libode-dev` - for ODE bindings

```sh
  git clone https://github.com/unn4m3d/crystal_sdl_test/
  cd crystal_sdl_test
  crystal deps
  crystal ./src/main.cr
```
